# CB Hugo - A modification of the Beautiful Hugo Theme

## Installation

    $ mkdir themes
    $ cd themes
    $ git clone https://github.com/clintonbale/cbhugo.git cbhugo

See [the Hugo documentation](http://gohugo.io/themes/installing/) for more information.
